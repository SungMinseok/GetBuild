
# QuickBuild 릴리즈 노트

**빌드 날짜**: 2025-10-26

## 변경사항

* 스케줄 진행 상태 실시간 표시 기능 추가
* ScheduleDialog에 소스/로컬 경로 설정 기능 추가
* 개별 스케줄 위젯에 진행 상태 표시
* 상단 헤더에 실행 중인 스케줄 요약 표시
* UI 개선 및 버그 수정

## 주요 기능

### 1. 스케줄 진행 상태 실시간 표시
- 각 스케줄 실행 시 진행 상태 시각적 표시
- 실행 중: 주황색 배경 + 진행 바
- 완료: 초록색 완료 메시지 (3초간 표시)
- 오류: 빨간색 오류 메시지 (5초간 표시)

### 2. 경로 설정 기능
- 스케줄 생성/편집 시 소스 경로와 로컬 경로 개별 설정 가능
- 폴더 찾아보기 버튼으로 쉬운 경로 선택

### 3. 상태 요약
- 상단 헤더에 현재 실행 중인 스케줄 개수 표시
- 실행 중인 스케줄 이름 미리보기 (최대 3개)

## 버그 수정
- 클라복사 실행 시 아무 일도 일어나지 않던 문제 수정
- 스케줄 실행 시 실제 파일 복사가 이루어지도록 수정
- 빌드명 자동 검색 기능 추가
