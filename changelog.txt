# QuickBuild 3.0.4 릴리즈 노트

**빌드 날짜**: 2026-02-13

## 변경사항

- 코드 정리: 불필요한 파일 대량 삭제 및 스크립트 경로 정리 (scripts/ 폴더)
- ChromeDriver 자동 관리: 버전 감지, 호환성 확인, 자동 다운로드/업데이트, 구버전 정리
- 서버업로드및패치: 패치 대기시간(분) 설정 기능 추가, 업로드 완료 시 1차 슬랙 알림
- 슬랙 알림: 스레드 댓글(채널에도 전송) 모드 추가 (thread_broadcast)
- 결과 파일을 result/ 폴더에 저장하도록 변경
- run.bat 개선: 가상환경 자동 감지 (.venv / pyenvs), 에러 처리 추가
