# QuickBuild 3.0.4 릴리즈 노트

**빌드 날짜**: 2026-02-13

## 변경사항

- 코드 정리 - 불필요한 파일 대량 삭제 및 스크립트 경로 정리
- ChromeDriver 자동 관리 (버전 감지, 호환성 확인, 자동 업데이트)
- 서버업로드및패치 - 패치 대기시간 설정 기능 추가
- 슬랙 알림 - 스레드 댓글(채널에도 전송) 모드 추가
- 결과 파일 result/ 폴더 저장, run.bat 가상환경 자동 감지

**자동 업데이트 지원**: QuickBuild를 실행하면 자동으로 새 버전을 확인합니다.
